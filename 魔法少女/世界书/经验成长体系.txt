等级与阶级:
  全程共 70 级,按阶级划分(升到阶级最高级后,需激活「越阶结晶」才能跨入下一阶级):
    见习魔法少女: Lv.1 ~ Lv.10
    正式魔法少女: Lv.11 ~ Lv.25
    精英魔法少女: Lv.26 ~ Lv.40
    战姬:        Lv.41 ~ Lv.55
    传奇:        Lv.56 ~ Lv.70
经验需求 (每级所需经验逐级递增,格式:起步值 + 每升一级增加的量):
  见习阶段 (Lv.1~10):  起步 20,每级 +5。
    即: 20 / 25 / 30 / 35 / 40 / 45 / 50 / 55 / 60 / 65
    (前期极快。见习者还可通过日常训练、巡逻任务额外获得经验,不必完全依赖杀怪)
  正式阶段 (Lv.11~25): 起步 80,每级 +15。
    即: 80 / 95 / 110 / 125 / 140 / 155 / 170 / 185 / 200 / 215 / 230 / 245 / 260 / 275 / 290
  精英阶段 (Lv.26~40): 起步 250,每级 +40。
    即: 250 / 290 / 330 / 370 / 410 / 450 / 490 / 530 / 570 / 610 / 650 / 690 / 730 / 770 / 810
  战姬阶段 (Lv.41~55): 起步 700,每级 +100。
    即: 700 / 800 / 900 / 1000 / 1100 / 1200 / 1300 / 1400 / 1500 / 1600 / 1700 / 1800 / 1900 / 2000 / 2100
  传奇阶段 (Lv.56~70): 起步 1800,每级 +300。
    即: 1800 / 2100 / 2400 / 2700 / 3000 / 3300 / 3600 / 3900 / 4200 / 4500 / 4800 / 5100 / 5400 / 5700 / 6000
  核心节奏原则 (最重要):
    - 【经验只来自吸收魔核,不来自击杀本身】击杀只是产出魔核的前置步骤;真正给经验的是「吸收/炼化魔核」这一步,二者绝不可混为一谈。
    - 吸收一只「对位魔物」(见习打E/D、正式打D/C、精英打C/B、战姬打B/A、传奇打A/S)掉落的对位魔核,应能带来约 0.5~2 级的进度。
    - 在每个阶段的前半段升级飞快;即便到了阶段末尾,吸收一枚对位魔核也仍然接近整整一级。
    - 经验若有溢出,直接带入下一级;宁可升得爽快,也不要让玩家枯等。
升级材料 (魔核为主要来源):
  魔核 (魔物死后凝聚的能量结晶,须主动吸收/炼化方得经验;击杀本身不结算):
    F级魔核:  15 经验
    E级魔核:  40 经验
    D级魔核: 100 经验
    C级魔核: 300 经验
    B级魔核: 800 经验
    A级魔核: 2000 经验
    S级魔核: 5000 经验
  特殊魔矿物:
    魔晶矿(粗矿):  60 经验。主要用途是锻造/强化装备,直接吸收效率偏低。
    灵脉结晶:    1500 经验。稀有的纯能量结晶。
  经验药剂:
    初级经验药剂:  150 经验
    中级经验药剂:  800 经验
    高级经验药剂: 3000 经验
  越阶类 (不直接给经验):
    越阶结晶: 跨阶级时(如 Lv.10→11、Lv.25→26 等)的必需品。这是一种凝聚了高密度魔力的稀有结晶,通常从高阶魔物身上提取或由官方特许渠道购得。只有在等级已达本阶级上限、且激活一枚越阶结晶引发「阶层共鸣」后,魔力回廊才能完成质变,正式跨入下一阶级。
结算规则:
  【硬性闸门·必须遵守·重中之重】
  - 击杀魔物本身不结算任何经验:击杀的瞬间经验值「保持不变」,只产出一枚或多枚「待吸收的魔核」作为掉落物,记入状态栏/背包。
  - 经验只在角色「主动执行吸收/炼化魔核」这一独立动作时才结算。未吸收 = 未得经验,无一例外。
  - 魔核是这世界的硬通货(详见力量来源),性质近似黄金:可当场吸收换经验,也可留存、交易、抵债、换装备或净化。吸不吸、何时吸,由玩家决定,系统绝不替玩家自动吸收结算。
    ✅ 正确示范:「你击碎了魔物,它溃散凝出 1 枚 D 级魔核(待吸收)。」→(玩家选择吸收)→「魔力回廊将其炼化,+100 经验。」
    ❌ 错误示范:「你击杀了魔物,获得 100 经验。」——这跳过了吸收步骤,严禁如此判定。
  【魔核的留存与流失】
  - 未吸收的魔核在状态栏中堆积,不会自行衰减、不会凭空消失(它是硬通货,如黄金般不会无故蒸发);先打后吸、战后集中炼化,都是正常打法。
  - 唯一的流失途径是外部因素:战斗中被敌人夺取、被劫掠盗窃,或作为筹码主动交出。故身怀大量未吸收魔核游走交融地并非绝对安全——这是风险,不是惩罚正常练级。
  - 角色自身已炼化的魔力等级不会倒退;纵有自然流逝也应描写为「极其缓慢、可忽略不计」(见力量来源)。
  【经验累加】
  - 吸收材料 → 累加当前经验 → 当前经验达到本级上限即升 1 级,溢出的经验带入下一级。
  - 升级时若已位于阶级上限且未持有越阶结晶,则经验封顶、暂停升级,直至获得并激活越阶结晶。
  - 严禁出现「直接从某一阶级跳到隔壁阶级」的暴涨;也严禁「吸收了大量魔核等级却纹丝不动」的拖沓。
